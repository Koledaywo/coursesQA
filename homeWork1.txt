1) Посмотреть где я  -- pwd
2) создать папку     -- mkdir Home1
3) Зайти в папку     -- cd Home1
4) Создать три папки -- mkdir Base Downloads Users
5) Зайти в любую папку -- cd Base
6) Создать 5 файлов -- touch 1.txt 2.txt 3.txt main.json back.json
7) Создать 3 папки -- mkdir 1 2 3
8) Вывести содержимое файла -- ls (если нужно вывести и скрытые файлы и файлы у которых перед названием стои точка, то добавляем -а: -l - выводить подробный список, в котором будет отображаться владелец, группа, дата создания, размер и другие параметры; можно вывести общий вариант прописав ls -la)
9) Открыть любой txt файл -- start 1.txt (nano 1.txt; vim 1.txt команды для открытия и редактирования файла в терминале)
10-11) Полсе открытия файла (например командой vim 1.txt) нажимае 'i', печатаем текстк, нажимаем esc, запем ':wq (команда для выходи а сохранения для vim, выйти без сохранения :q!)
12) Выйти из папки на уровень выше -- cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку -- mv main.json 1/main.json
mv 2.txt 1/2.txt (интересный момент, что при перемещении и копировании мы можем изменить название файла и его расширениее. например можно написать mv 2.txt 1/2.docx таким образом мы переместили бы файл, поменяв его расширение)
14)скопировать любые 2 файла, которые вы создали, в любую другую папку --
cp 1.txt 2/1.txt
cp back.json 2/back.json
15) Найти файл по имени -- find -name 1.txt  (-name - по имени файла)
16) просмотреть содержимое в реальном времени -- cat 1.txt
17) вывести несколько первых строк из текстового фай —— head -2 1.txt
18) вывести несколько последних строк из текстового файла —
tail -2 1.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.  cat 1.txt ( less 1.txt)
20) Вывести дату и время -- date
